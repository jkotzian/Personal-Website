Naval Warfare Simulator

Source code and documentation

        This is a naval warfare simulator that I created for a project in my Advanced Object Oriented Design class. While it may not look like much, there's a lot going on under the hood here. Users can create ships and move them in a direction or to a specific island. Ships require fuel to move, but they can refuel at the islands. Cruisers can attack other ships, while tankers can transport fuel to other islands. The powerful and autonomous pirate ship will move towards and attack the nearest ship! Fleets can also be created, allowing the user to command multiple ships at a time.

        The main focus of this project was to write readable, and extensible code. We took advantage of design patterns to accomplish this, like the Template and Composite/Component patterns.
